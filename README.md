# 🛒 Htet_X E-Commerce Platform

A modern full-stack e-commerce web application built with React and Supabase, designed to provide a smooth and responsive online shopping experience.

---

## 🚀 Project Overview

Htet_X is an e-commerce platform that allows users to browse products, view details, manage cart items, and proceed through checkout flow.

The project is currently under active development. User-side features are mostly completed, while admin panel and payment system are in progress.

---

## ✨ Features

### 👤 User Features

- User authentication (Sign up / Login)
- Product listing with categories
- Product detail page
- Add to cart / remove from cart
- Quantity management
- Responsive UI (mobile-friendly)

### 🧠 System Features

- Supabase backend integration
- Real-time database support
- Secure authentication handling

### 🚧 In Progress

- Payment system integration
- Admin dashboard (product & order management)
- Order tracking system

---

## 🛠️ Tech Stack

- React + TypeScript
- Tailwind CSS
- Supabase (Database + Auth + API)
- React Router
- Vite
